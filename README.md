**本项目为嘲讽国内媒体无下线的炒作, 包括但不限于 央视网、大象新闻**

**以及没有明辨是非能力并煽动网暴的人**

## 使用

- 上帝公式

```
  JPMath.godFormula("主"); // 6
```

- 姜指变换

```
  JPMath.jpPow(2, 3); // 2^3 = 2 * 3 = 6
```

- 二分求和

```
  JPMath.jpSigma("Σ"); // ∞ / 2
```

## 将此库运用到你的项目中

### Gradle

从jitpack获取

```
repositories {
    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.Crystal0404.JumpingMath:<version>'
}
```
